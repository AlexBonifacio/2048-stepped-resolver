2048 Helper - version Windows
================================

Pour lancer:

1. Dezippe le dossier complet.
2. Double-clique sur:

   Lancer 2048 Helper.bat

3. Une page web s'ouvre.
4. Garde la fenetre noire ouverte pendant toute la partie.
5. Pour arreter, ferme la fenetre noire.

Si Windows affiche une alerte:

- Autorise l'application sur le reseau prive/local.
- L'application tourne sur ton ordinateur uniquement: http://127.0.0.1:8765

Si le fichier dit que Python est introuvable:

- Tu n'as pas la version complete avec Python embarque.
- Demande le zip complet, ou installe Python 3 depuis https://www.python.org/downloads/windows/

Si le fichier dit que 2048-ranks.exe est introuvable:

- Le zip n'est pas complet.
- Demande une version Windows complete.

Les parties sont sauvegardees dans:

data\sessions\

La session ouverte par defaut s'appelle:

ma_partie
